# gdb-win_x86-64

## 概要

Raspberry PiなどのコードをWindowsで動作するvscodeから、 gdbserver経由でクロスデバッグするためのバイナリです。 

------

## ライセンス

ソースコード側に記載(GPL2)

------

## ソースコードの場所

http://ftp.jaist.ac.jp/pub/GNU/gdb/

------

## ビルド

msys2でx86_64-w64-mingw32を使用してビルドを行っています。
Windows10 x86_64をターゲットにしています。

ビルドを行ため、一部のソースファイルに修正を行っています。
